## Scope

Implement **v0.3.0 P0 only** — scorer + tier classifier + staffing dispatcher + two-step CEO flow. P1 (expert auto-summoning) and P2 (polish) are DEFERRED to v0.3.1 / v0.3.2 respectively.

## CRITICAL — consultant API decision

**Per audit memo `consultant-api-audit-2026-05-18.md`**: `aiplus-auto-team-consultant` Coordinator API is NOT stable for direct reuse (scale conflict + no code-level API + pre-1.0 module).

**Implement scoring INTERNALLY** in `crates/aiplus-cli/src/agent/coordinator.rs` per DESIGN.md §9.1 spec (1-5 complexity + 0.0-1.0 risk). Do NOT import from `aiplus-auto-team-consultant`. Document this deviation from DESIGN.md §4 DRY in IMPLEMENTATION notes; justified by audit.
